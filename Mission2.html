Mission 2

Mission 2 required us to use our satellite to receive a message from a space craft



We looped through each radio channel and if the channel received anything then the screen would light up in a hearts display



From there, once we knew the radio channel we reprogrammed the satellite to read the message that it sent. JEJAXUPDENPU



We can rule out any keyword ciphers to decrypt the message based on the face there was no key. The most likely option we could think of was a simple Caesar cipher. This is because if you analyse the frequency of the letters we can deduce that J and J come from the same plaintext letter. Therefore we can rule out many of the uncommon letters like {Q,W,Y,H,J(through the rarity of if),K,X,}.From there we brute forced the decryption



We set up two for loops to iterate through the string, first we set up a for loop that iterates 25 times(number of possible combinations), from there we set up another for loop this time the length of the string received. It iterated through each plaintext letter and applied a shift of value k to give the codedletter. The value of k was part of the first for loop. To make sure that the program looped through the alphabet and didn't run off past the alphabet, we converted each letter to its ASCII equivalent by taking its place in the alphabet {i.e. A is 1} and then adding 64 to it{A becomes 65}. Then if the shift meant the value was over Z's value {90} we simply minused 25.



This then gave us the decoded message of NINEBYTHIRTY which we used the Egyptian base 10 system to times 9 by 30 to give 270. We transmitted this back to them big box and watched as it became a tick
